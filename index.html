API Documentation

Integrate SnapDownloader into your applications

GET /api/snap-down?url=

Description

Download Snapchat videos programmatically

Parameters

url Snapchat video URL (required)

Example Request

fetch('https://your-site.netlify.app/api/snap-down?url=https://www.snapchat.com/spotlight/W7_EDlXWTBiXAEEniNoMPwAAYaXlpZnh2b2tnAZzb_v5RAZzb_v41AAAAAQ')
    .then(response => response.json())
    .then(data => console.log(data));

Response Format

{
    "success": true,
    "videoUrl": "https://cdn.snapchat.com/video.mp4",
    "thumbnail": "https://cdn.snapchat.com/thumbnail.jpg",
    "title": "Video Title",
    "duration": "00:15"
}
