API Documentation

Integrate SnapDownloader into your applications with our simple REST API

GET
/api/snap-down

Download Snapchat videos programmatically

Parameters

url string Snapchat video URL (required)

Example Request

fetch('https://your-api.netlify.app/api/snap-down?url=YOUR_SNAPCHAT_URL')
    .then(response => response.json())
    .then(data => console.log(data));

Response Format

{
    "success": true,
    "videoUrl": "https://cdn.snapchat.com/video.mp4",
    "thumbnail": "https://cdn.snapchat.com/thumb.jpg",
    "title": "Video Title",
    "duration": "00:15",
    "quality": "HD"
}
